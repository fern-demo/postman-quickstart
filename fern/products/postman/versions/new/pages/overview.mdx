The Postman API lets you programmatically access data stored in your Postman account. Use it to manage collections, environments, workspaces, and more.

## Base URL

All API requests use the following base URL:

```
https://api.getpostman.com
```

For EU users on Enterprise plans, use `https://api.eu.postman.com` instead.

## Key features

- Manage collections, environments, and workspaces
- Automate API testing workflows
- Access monitoring and mock server data
- Integrate Postman into your CI/CD pipeline
